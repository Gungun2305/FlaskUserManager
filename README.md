# 🧑‍💻 Flask User API

A simple RESTful API built using **Flask** that performs basic CRUD operations on in-memory user data.


## 🚀 Features

-  Create a new user
-  Retrieve all users or a specific user
-  Update existing user info
-  Delete a user

---

## Requirements

- Python 3.x
- Flask

Install Flask using pip:
```bash
pip install flask
